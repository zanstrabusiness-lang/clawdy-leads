Clawdy Leads Preview Hub

This GitHub Pages site hosts the personalized preview homepages we craft for local businesses. Each lead gets its own slug folder with a static mockup URL, so we can drop links directly into outreach messages.

How to publish a preview

  1. Run the automation in leads/ to gather business info.
  2. Render a homepage into previews/<slug>/index.html using the frontend skill.
  3. Commit the previews/ directory and push to main.
  4. GitHub Pages automatically publishes it at https://zanstrabusiness-lang.github.io/clawdy-leads/previews/<slug>/.

If the business owns a custom domain, drop a CNAME file inside the slug folder and point their DNS to GitHub Pages—then the preview is accessible at their real URL while still matching the slug path.
